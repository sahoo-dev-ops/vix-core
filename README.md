## Overview

VIX (Vehicle Intelligence X) is a rule-based, signal-driven vehicle
risk assessment and decision engine designed for lending,
insurance, and valuation use cases.

The system focuses on:
- Explainability (bank & regulator friendly)
- Deterministic decision logic
- Gradual evolution toward ML-based intelligence

This repository represents **Phase 1** of the VIX system.
